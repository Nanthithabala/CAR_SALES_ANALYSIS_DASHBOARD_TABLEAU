# CAR_SALES_ANALYSIS_DASHBOARD_TABLEAU

End-to-end Tableau dashboard analyzing car sales performance, regional trends, transmission preferences, and vehicle body style distribution.

## Project Overview

This Tableau dashboard provides comprehensive insights into automotive sales performance through interactive visualizations and business intelligence techniques. The dashboard enables users to analyze sales trends, dealer performance, vehicle preferences, and top-performing models.

## Dashboard Objectives

- Monitor overall sales performance
- Analyze company-wise sales quantity
- Compare dealer region performance
- Study transmission preferences
- Identify top-selling vehicle models
- Explore body style distribution

## Key Metrics

- **Total Sales Quantity:** 17,277
- **Total Sales:** $480.4M
- **Automatic Cars Sold:** 9,273 (53.67%)
- **Manual Cars Sold:** 8,004 (46.33%)

## Dashboard Features

- Interactive Filters (Company Name & Order Date)
- KPI Cards
- Company Sales Insights
- Dealer Region Analysis
- Auto vs Manual Sales Comparison
- Top 20 Models Analysis
- Cumulative Sales Analysis
- Transmission Flow Analysis
- Body Style Distribution

## Visualizations Used

- KPI Cards
- Lollipop Chart
- Donut Chart
- Dumbbell Chart
- Pareto Chart
- Sankey Diagram
- Treemap
- Interactive Filters

## Business Insights

- Automatic transmission vehicles account for over 53% of total sales.
- SUV and Hatchback body styles generate the highest revenue.
- Sales performance varies significantly across dealer regions.
- A small group of vehicle models contributes a large share of total sales.
- Company-wise analysis reveals differences in market performance among automobile brands.
